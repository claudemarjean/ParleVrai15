# ParleVrai15

Apprendre à parler français en 15 minutes par jour.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Architecture

- **Vanilla JavaScript** avec Vite
- **Mobile-first** responsive design
- **Modulaire** et scalable
- Préparé pour intégration **Supabase**

## Structure

```
/src
  /pages          # Pages de l'application
  /components     # Composants réutilisables
  /styles         # CSS global et modules
  /services       # Services (API, auth, etc.)
  /utils          # Utilitaires
  /router         # Système de routing
```

## Fonctionnalités

- Landing page publique
- Authentification (inscription/connexion)
- Dashboard utilisateur
- Leçon du jour (lecture, grammaire, vocabulaire, exercice oral)
- Calendrier de progression
- Statistiques
- Back-office admin (CRUD leçons)
